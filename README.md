# Sales Data EDA using Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a retail sales dataset using Python. The objective is to understand customer purchasing behavior, identify top-performing customer segments, analyze sales trends, and generate business insights from the data.

The analysis was performed using Pandas, Matplotlib, and Seaborn in Jupyter Notebook.

---

## Project Objectives

* Analyze customer purchasing behavior.
* Identify high-value customer segments.
* Find top-performing states.
* Analyze product category performance.
* Understand profession-wise purchasing patterns.
* Identify best-selling products.
* Generate business recommendations based on data.

---

## Dataset Information

The dataset contains retail sales transaction records including:

* Customer Gender
* Age Group
* State
* Profession
* Product Category
* Quantity Ordered
* Sales Amount

## Data Cleaning Performed

* Imported dataset using Pandas
* Checked data structure and data types
* Removed unnecessary columns
* Identified and removed null values
* Removed duplicate records
* Converted data types where required
* Prepared data for analysis

---

## Exploratory Data Analysis Questions

This project answers the following business questions:

* Which gender contributes the highest number of transactions and sales?
* Which age groups generate the most revenue?
* Which states contribute the highest sales and order quantities?
* Which product categories perform best in terms of transactions and sales amount?
* How do purchasing patterns differ across genders and product categories?
* Which professions contribute the most transactions and revenue?
* Which products have the highest order quantities?

## Key Insights

* Female customers contribute higher sales compared to male customers.
* Customers aged 26–35 generate the highest sales and transaction volume.
* Uttar Pradesh, Maharashtra, and Karnataka are among the top-performing states.
* Beauty, Sports, and Electronics categories contribute significantly to overall sales.
* Customers working in IT, Healthcare, and Aviation sectors generate strong revenue.
* Several products consistently achieve high order quantities and can be considered best sellers.

---

## Business Recommendations

* Focus marketing campaigns on female customers.
* Develop targeted promotions for customers aged 26–35.
* Increase inventory and marketing efforts in top-performing states.
* Prioritize high-performing product categories.
* Design profession-based promotional campaigns for key customer groups.
